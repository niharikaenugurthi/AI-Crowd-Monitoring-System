# AI Crowd Monitoring System

An AI-powered crowd monitoring system that detects people in real-time using computer vision and displays crowd analytics through a live web dashboard.

## Project Overview

The AI Crowd Monitoring System uses deep learning to detect and count people in a video stream. The system helps monitor crowd density in real time and generates alerts when overcrowding is detected.

It can be used in places like:

- Public events
- Shopping malls
- Railway stations
- Airports
- Smart city surveillance systems

The system streams live video, counts people automatically, displays crowd density levels, and visualizes crowd trends on a dashboard.

---

## Features

- Real-time people detection
- Live camera video streaming
